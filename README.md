# Calculator React

---

### A simple calculator made with React for Orange Tech+ from Banco Inter's Bootcamp.

![Default](/assets/default.png)
![History](/assets/history.png)

---

## Clone project

Required:
- Node >=v16.14.1
- Git

```
   $ git clone https://github.com/WesleyMaik/calculator-react
```

---

## Install

```
npm install
```
or
```
yarn install
```

---

## Run

```
npm dev
```
or
```
yarn dev
```
